Html tags

Heading Tags

       
<h1>Heading 1</h1> 
<h2>Heading 2</h2>
<h3>Heading 3</h3> 
<h4>Heading 4</h4>
<h5>Heading 5</h5> 
<h6>Heading 6</h6>

These tags are used  to create headings in your HTML document.
h1 represents the most important heading while h6 represents the least important heading.

Paragraph Tag

       
<p>This is a paragraph.</p>

This tag is used to create paragraphs of text.

Link Tag

       
<a href="https://www.youtube.com">Link Text</a>

This tag is used to create links to other web pages or resources. The href attribute specifies the URL of the target resource
 and the text between the opening and closing tags is what the user clicks on to follow the link.

Image Tag

       
<img src="robot.png" alt="image description">

This tag is used to insert images into your web page. The src attribute specifies the URL of the image file, while the alt 
attribute provides alternative text for screen readers or if the image cannot be displayed.

List Tags

Unordered List
        
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

It is used to create an unordered list of items. The list items are preceded by bullet points, which are automatically
generated by the browser.
Ordered List
        
<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>

It is used to create an ordered list of items. The list items are numbered sequentially by the browser.

Subscript Tag

       
<sub>Text</sub>

It is used to create subscript text in your HTML document where the text is placed below the baseline of the rest of the text.
It is commonly used in scientific notation, mathematical formulas, and chemical equations.

Superscript Tag

       
<sup>Text</sup>

It is used to create superscript text in your HTML document where the text is placed above the baseline of the rest of the text.
It is also commonly used in scientific notation, mathematical formulas, and chemical equations.

Blockquote Tag

       
<blockquote>Text</blockquote>

This tag is used to create a blockquote, which is a quotation from another source that is presented as a separate block of text.

Bold Tag

       
<b>Text</b>

This tag is used to bold text, which makes it stand out from the surrounding content.

Css properties

Background Color

The background-color property sets the background color of an element.

This is a box with a yellow background color.

          
            .box {
              background-color: yellow;
            }

Color

The color property specifies the color of the text.

This is a box with blue text color.

          
            .box {
              color: blue;
            }

Font Size

The font-size property sets the font size of an element's text content.

This is a box with larger font size text.

          
            .box {
              font-size: 32px;
            }

Font Family

The font-family property sets the font family of an element's text content.

This is a box with Times New Roman font.

          
            .box {
              font-family: 'Times New Roman', serif;
            }

Text Align

The text-align property sets the horizontal alignment of an element's text content.

This is a box with right-aligned text.

          
            .box {
              text-align: right;
            }

Padding

The padding property sets the padding between an element's content and its border.

This is a box with extra padding.

          
            .box {
              padding: 50px;
            }

Margin

The margin property in CSS is used to define the amount of space around an HTML element, outside of its border.

This is a box with a margin of 30px.

          
            .box {
              margin: 30px;
            }

Height

The height property in CSS is used to set the height of an element, which determines the vertical space it takes up on a web page.

This is a box with a height of 90px.

          
            .box {
              height: 90px;
            }

width

The width property in CSS is used to set the width of an element, which determines the horizontal space it takes up on a web page.

This is a box with the width of 70px.

          
            .box {
              width: 70px;
            }
